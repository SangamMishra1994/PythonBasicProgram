# FastAPI Tutorial Project

This is a FastAPI backend project built for learning and practice.

## Tech Stack

- Python 3.10+
- FastAPI
- Uvicorn
- Pipenv

## Setup Instructions

1. [ ] Clone the repository
2. [ ] Create virtual environment
3. [ ] Install dependencies
4. [ ] Run the application

## Run Project

```bash
pipenv install
pipenv run uvicorn main:app --reload
```
