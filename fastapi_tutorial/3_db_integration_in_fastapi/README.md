# FastAPI Tutorial Project

This is a FastAPI backend project built for learning and practice.

## Tech Stack

- Python 3.10+
- FastAPI
- Uvicorn
- Pipenv

## Setup Instructions

1. [x] Intro: DB integration in fastapi
2. [ ] Connecting our API to a real database and using ORM tools like SQLModel.

```bash

```
